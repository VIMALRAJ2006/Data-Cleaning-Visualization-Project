# Data Cleaning & Visualization Project

## Objective
This project focuses on cleaning and visualizing retail sales data using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Tasks Performed
- Data cleaning
- Duplicate removal
- Date conversion
- Statistical analysis
- Data visualization

## Visualizations
- Sales by Category
- Profit by Region
- Correlation Heatmap
- Sales Distribution

## Conclusion
The project identifies sales trends and profit patterns from retail data.
